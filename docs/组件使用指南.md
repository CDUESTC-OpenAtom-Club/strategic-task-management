## 共享组件

### 数据展示组件

#### DataTable

通用数据表格组件，支持分页、排序、筛选。

```vue
<template>
  <DataTable
    :data="tableData"
    :columns="columns"
    :loading="loading"
    :pagination="pagination"
    @page-change="handlePageChange"
    @sort-change="handleSortChange"
  />
</template>

<script setup lang="ts">
import { DataTable } from '@/shared/ui/table'

const columns = [
  { prop: 'name', label: '名称', sortable: true },
  { prop: 'status', label: '状态', sortable: true },
  { prop: 'createdAt', label: '创建时间', sortable: true }
]

const tableData = ref([])
const loading = ref(false)
const pagination = reactive({
  page: 1,
  pageSize: 20,
  total: 0
})
</script>
```

**Props**:

- `data` - 表格数据
- `columns` - 列配置
- `loading` - 加载状态
- `pagination` - 分页配置

**Events**:

- `page-change` - 页码变化
- `sort-change` - 排序变化
- `selection-change` - 选择变化

动态表单组件，基于配置生成表单。

```vue

</template>

<script setup lang="ts">

  {
    field: 'name',
    label: '名称',

    field: 'type',
    label: '类型',
    type: 'select',
    options: [
      { label: '类型A', value: 'a' },
      { label: '类型B', value: 'b' }

]

  name: '',
  type: ''
})

  console.log('表单提交:', data)
```

```vue
</script>
```

## 异步组件

### 使用 defineAsyncComponent

对于大型组件，使用异步加载：

```vue
<script setup lang="ts">
import { defineAsyncComponent } from 'vue'

// 基础用法
const HeavyComponent = defineAsyncComponent(() => import('@/components/HeavyComponent.vue'))

// 带加载状态
const AdvancedChart = defineAsyncComponent({
  loader: () => import('@/components/AdvancedChart.vue'),
  loadingComponent: SkeletonLoader,
  errorComponent: ErrorComponent,
  delay: 200,
  timeout: 3000
})
</script>
```

### 使用 createAsyncComponent 工具

```vue
<script setup lang="ts">
import { createAsyncComponent } from '@/utils/asyncComponent'

const HeavyComponent = createAsyncComponent(() => import('@/components/HeavyComponent.vue'), {
  delay: 200,
  timeout: 10000,
  maxRetries: 2,
  showSkeleton: true
})
</script>
```

```typescript
// router/index.ts
const routes = [
  {
    path: '/dashboard',
    component: () => import('@/pages/dashboard/ui/DashboardPage.vue')
  }
```

### 常用组件示例

#### Dialog

```vue
<template>
  <el-dialog v-model="visible" title="对话框">
    <span>内容</span>
    <template #footer>
      <el-button @click="visible = false">取消</el-button>
      <el-button type="primary" @click="handleConfirm">确定</el-button>
    </template>
  </el-dialog>
</template>

const handleConfirm = () => {
  visible.value = false
}
</script>
```

#### Form

```vue
<template>
  <el-form :model="form" :rules="rules" ref="formRef">
    <el-form-item label="名称" prop="name">
      <el-input v-model="form.name" />
    </el-form-item>
    <el-form-item label="类型" prop="type">
      <el-select v-model="form.type">
        <el-option label="类型A" value="a" />
        <el-option label="类型B" value="b" />
      </el-select>
    </el-form-item>
  </el-form>
</template>

<script setup lang="ts">
import { reactive, ref } from 'vue'
import type { FormInstance, FormRules } from 'element-plus'

const formRef = ref<FormInstance>()
const form = reactive({
  name: '',
  type: ''
})

const rules = reactive<FormRules>({
  name: [{ required: true, message: '请输入名称', trigger: 'blur' }],
  type: [{ required: true, message: '请选择类型', trigger: 'change' }]
})
</script>
```

## 性能优化建议

### 1. 组件懒加载

**适用场景**:

- 大型图表组件（ECharts）
- 复杂表单组件
- 条件渲染的组件
- 弹窗/对话框内容

**示例**:

```vue
<script setup lang="ts">
import { defineAsyncComponent } from 'vue'

// ✅ 推荐：大型组件懒加载
const HeavyChart = defineAsyncComponent(() => import('@/components/HeavyChart.vue'))

// ❌ 不推荐：小型组件直接导入
import { SimpleButton } from '@/components/SimpleButton.vue'
</script>
```

### 2. v-show vs v-if

```vue
<template>
  <!-- ✅ 频繁切换：使用 v-show -->
  <div v-show="isVisible">内容</div>

  <!-- ✅ 条件很少改变：使用 v-if -->
  <div v-if="isLoggedIn">用户内容</div>
</template>
```

### 3. 列表渲染优化

```vue
<template>
  <!-- ✅ 使用 key 帮助 Vue 识别节点 -->
  <div v-for="item in list" :key="item.id">
    {{ item.name }}
  </div>

  <!-- ✅ 避免使用 index 作为 key（除非列表是静态的） -->
  <div v-for="(item, index) in staticList" :key="index">
    {{ item }}
  </div>

  <!-- ✅ 大列表使用虚拟滚动 -->
  <VirtualList :items="largeList" :item-size="50" />
</template>
```

### 4. 计算属性缓存

```vue
<script setup lang="ts">
import { computed } from 'vue'

// ✅ 使用 computed 缓存计算结果
const filteredList = computed(() => {
  return list.value.filter(item => item.active)
})

// ❌ 避免在模板中使用复杂表达式
// <template>
//   {{ list.filter(item => item.active).map(item => item.name) }}
// </template>
</script>
```

### 5. 事件监听器清理

```vue
<script setup lang="ts">
import { onMounted, onUnmounted } from 'vue'

onMounted(() => {
  window.addEventListener('resize', handleResize)
})

onUnmounted(() => {
  // ✅ 记得清理事件监听
  window.removeEventListener('resize', handleResize)
})
</script>
```

### 6. 组件通信优化

```vue
<script setup lang="ts">
// ✅ 使用 provide/inject 避免多层 props 传递
// 父组件
provide('theme', theme)

// 子组件
const theme = inject('theme')

// ✅ 使用事件总线或 Pinia 管理跨组件状态
import { useEventBus } from '@/composables/useEventBus'
const bus = useEventBus()
bus.emit('event-name', data)
</script>
```

## 组件开发规范

### 1. 命名规范

- **组件文件**: PascalCase（如 `UserList.vue`）
- **组件引用**: PascalCase（如 `<UserList />`）
- **Props**: camelCase（如 `userName`）
- **Events**: kebab-case（如 `@user-update`）

### 2. 组件结构

```vue
<template>
  <div class="component-name">
    <!-- 模板内容 -->
  </div>
</template>

<script setup lang="ts">
// 1. 导入
import { ref, computed } from 'vue'

// 2. Props 定义
interface Props {
  modelValue: string
  disabled?: boolean
}
const props = withDefaults(defineProps<Props>(), {
  disabled: false
})

// 3. Emits 定义
interface Emits {
  (e: 'update:modelValue', value: string): void
  (e: 'change', value: string): void
}
const emit = defineEmits<Emits>()

// 4. 响应式数据
const localValue = ref('')

// 5. 计算属性
const computedValue = computed(() => {
  return props.modelValue.toUpperCase()
})

// 6. 方法
const handleChange = () => {
  emit('update:modelValue', localValue.value)
  emit('change', localValue.value)
}

// 7. 生命周期
onMounted(() => {
  // 初始化逻辑
})
</script>

<style scoped lang="scss">
.component-name {
  // 样式
}
</style>
```


4. **Slots 说明** - 插槽列表
5. **使用示例** - 代码示例

```vue
<!--
```
