## 功能流程

### 审批流程图

```
用户点击"审批"按钮
    ↓
打开计划审批抽屉
    ↓
加载待审批列表 (GET /plans/approval/pending)
    ↓
显示审批卡片
    ↓
用户选择操作
    ├─ 审批通过
    │   ↓
    │   输入审批意见（可选）
    │   ↓
    │   调用 approvePlan() API
    │   ↓
    │   显示成功消息
    │   ↓
    │   刷新列表和数据
    │
    └─ 审批拒绝
        ↓
        输入拒绝原因（必填）
        ↓
        调用 rejectPlan() API
        ↓
        显示成功消息
        ↓
        刷新列表和数据
```

### 数据流

```
PlanApprovalDrawer
    ↓ (加载)
approvalApi.getPendingApprovals()
    ↓ (返回)
pendingApprovals[]
    ↓ (渲染)
审批卡片列表
    ↓ (操作)
approvalApi.approvePlan() / rejectPlan()
    ↓ (成功)
emit('refresh')
    ↓
StrategicTaskView.handleApprovalRefresh()
    ↓
刷新待审批数量 + 刷新指标数据
```
